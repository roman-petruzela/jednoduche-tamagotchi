# rpprojekt

Jednoducha formulova aplikace s tamagoci.

## Co program dela
- Pridava mazlicky do seznamu
- U kazdeho mazlicka sleduje hlad, naladu, energii a vek
- Muzes mazlicka krmit, hrat si s nim, nechat ho spat a restartovat ho
- Cas ve hre plyne po minutach

## Jak spustit
- Otevri `rpprojekt.sln` ve Visual Studiu
- Spust projekt pres `Start`

## Pouzite prvky
- `TextBox`
- `Button`
- `ListBox`
- `TrackBar`
- `ProgressBar`
- `Timer`
